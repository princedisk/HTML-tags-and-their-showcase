Tags Explained

  • <!DOCTYPE> → Defines the document type (HTML5).
  • <html> → Root of the HTML document.
  • <head> → Container for metadata (not shown directly).
  • <title> → Sets the document title (browser tab text).
  • <body> → Holds all visible page content.

Metadata Explained

  • <base>:- google is a search engine.
  • <Header>:-Introductory content at the top.
  • <link>:-The link used to link stylesheets or external resource. Example: link rel="stylesheet" href="styles.css"
